Go pprof Endpoints

Standard runtime profiling provided by net/http/pprof. Note: Viewing these requires local authentication or `go tool pprof`.

  • /debug/pprof/ Main pprof HTML Index
  • 🔥 Interactive CPU Flamegraph (5s) Open CPU profile in Speedscope
  • /debug/pprof/profile (5s) Download raw 5s CPU profile (for go tool pprof)
  • 🔥 Interactive Heap Flamegraph Open Heap profile in Speedscope
  • /debug/pprof/heap Download raw heap allocation profile
  • 🔥 Interactive Goroutine Flamegraph Open Goroutine profile in Speedscope
  • /debug/pprof/goroutine?debug=1 View raw text stack traces of all current goroutines
  • /debug/pprof/trace (5s) Download execution trace (for go tool trace)
  • 🔥 Interactive Block Flamegraph Open Block profile in Speedscope
  • 🔥 Interactive Mutex Flamegraph Open Mutex profile in Speedscope
